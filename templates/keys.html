MACCS 166-Bit Molecular Fingerprints

A standardized cheminformatics descriptor system used to encode molecular structure into binary features for machine learning and chemical informatics models.

What are MACCS Keys?

MACCS (Molecular ACCess System) keys are a set of predefined structural fragments used to represent molecules as a fixed-length binary fingerprint (166 bits).

Each bit corresponds to a specific chemical substructure or property. If the feature exists in a molecule, the bit is set to 1; otherwise, it is 0.

These fingerprints are widely used in:

  • Drug discovery
  • QSAR modeling
  • Herbicide / pesticide prediction models
  • Molecular similarity search
