How to Use the Model

A quick guide to performing herbicide predictions

Single Prediction using Molecule Editor

  • Use the Draw Molecule panel (Ketcher) on the Home page to draw your chemical structure.
  • Once drawn, click Transfer to Input →. This converts your drawing into a SMILES string.
  • Click Run Prediction.
  • The model will display the prediction, confidence score, explanation charts, and molecular properties.

Single Prediction using SMILES

  • Paste a SMILES string into the SMILES Identifier text box.
  • Click Run Prediction or press Enter.

PubChem CID Lookup

  • Enter a PubChem Compound ID (CID) in the lookup section and click Fetch & Predict.
  • The SMILES will be retrieved from PubChem, filled in automatically, and the prediction will run immediately.
  • Find CIDs at pubchem.ncbi.nlm.nih.gov.

Batch Prediction

  • Switch to the Batch Prediction tab.
  • Paste SMILES strings, one per line.
  • Click Run Batch Prediction.
  • Download results as CSV using the Download CSV button.

Understanding and Interpreting the Results

  • Feature Contributions - the DONUT chart shows which MACCS structural keys had the most impact on the prediciton. The chart is color coded as green for fingerprints supporting herbicidal prediction and red for fingerprints supporting not-herbicide prediction.
  • Molecular Properties Profile - Radar chart of key descriptors (molecular weight, LogP, H-bond donors/acceptors, TPSA, rotatable bonds).
  • Fingerprint Evidence Classification - visual representation of the fingerprints associated with teh prediction.
  • Evidence table - Table shows the important fingerprints ranked in orde of feature importance and the effect of the fingerprint on herbicide prediciton.
  • Association shows the effect of the fingerprint in realtion to the herbicidal activity.
  • Status shows the effect of the fingerprint in the input compound.
